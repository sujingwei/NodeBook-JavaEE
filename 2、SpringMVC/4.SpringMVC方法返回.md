# Spring MVC 方法返回值

### 1、返回字符串(jsp视图)

```java
@RequestMapping("/testString")
public String testString(ModelMap model){
    System.out.println("testString方法执行了...");
    // 模拟读数据库
    User user = new User();
    user.setUsername("美美");
    user.setPassword("123");
    user.setAge(30);
    model.addAttribute("user", user);
    return  "success";
}
```

### 2、返回值是void类型

```java
/**
 * 返回值为 void
 */
@RequestMapping("/testVoid")
public void testVoid(HttpServletRequest request, HttpServletResponse response) 
    throws ServletException, IOException 
{
    System.out.println("testVoid方法执行了...");
    // 要么请求转发，要么请求重定向，要么直接响应浏览器输出
    // request.getRequestDispatcher("/WEB-INF/pages/success.jsp")
    // 	.forward(request, response);
    response.setContentType("text/html;charset=UTF-8");
    response.setCharacterEncoding("UTF-8");
    response.getWriter().print("hello 你好");
    return;
}
```

### 3、ModelAndView

```java
/**
 * 返回 ModelAndView 对象
 * @return
 */
@RequestMapping("/testModelAndView")
public ModelAndView testModelAndView(){
    ModelAndView mav = new ModelAndView();
    // 模拟读数据库
    User user = new User();
    user.setUsername("小凤");
    user.setPassword("456");
    user.setAge(30);
    mav.addObject("user", user);
    mav.setViewName("success");
    return mav;
}
```

### 4、请求转发和重定向

```java
/**
 * 请求转发和重定向
 * @return
 */
@RequestMapping("/testForwardOrRedirect")
public String testForwardOrRedirect(){
    System.out.println("testForwardOrRedirect方法执行了...");
    // 请求转发
    // return "forward:/WEB-INF/pages/success.jsp";

    // 重定向
    return "redirect:/index.jsp";
}
```

### 5、@ResponseBody响应json数据

***java代码***

```java
/**
 * 方法返回json数据
 * @param user
 * @return
 */
@RequestMapping(value = "/testAjax")
public @ResponseBody User testAjax(@RequestBody User user){
    System.out.println("testAjax方法执行了...");
    System.out.println(user);
    user.setUsername("haha");
    user.setAge(40);
    // 响应
    return user;
}
```

***前端代码***

```js
$(function(){
    $('#btn').click(function () {
        $.ajax({
            url: 'user/testAjax',
            type: 'POST',
            contentType: "application/json;charset=utf-8",
            // 把json数据转为字符串
            data: JSON.stringify({"username":"hehe","password":"123","age":30}),
            dataType: "JSON",
            success:function (data) {
                console.log(data);
            }
        });
    });
});
```
