# 一、栈

栈，就在程序在运行过程中<i style="color:deeppink;">用于存放线程的局部变量的内存空间</i>。<u>它会给每一个线程分配一个不同的空间</u>。

![](http://notebook-1.aoae.top/15992905412781)

栈桢，每个方法对应的内存区域就是栈桢。

# 二、方法区

方法区主要存放常量、静态变量、类元信息等数据。如果静态变量也是通过new得到的一个对象，那么这个静态变量最终的值也是在堆里得到的。
